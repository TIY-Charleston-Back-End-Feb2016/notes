  * Physical layer (hardware)
    * Wireless
    * Electrical (ethernet, coaxial)
    * Fiber optics
  * Internet layer (packets)
    * IPv4
    * IPv6
  * Transport layer (reliability)
    * UDP (checksum)
    * TCP (checksum and retransmission)
  * Application layer (end users)
    * ASCII
    * UTF-8
    * HTTP
    * HTTP/2
